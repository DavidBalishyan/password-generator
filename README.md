# Password generator

Yet another password generator website. Nothing serious, just a website I've built during my free time to revisit react, since I haven't done any react in quite some time

# Build instructions

**Install dependencies:** `pnpm install`
**Dev server:** `pnpm dev`
**Build for prod:** `pnpm build`

*Or use npm, yarn, etc... whatever you'd like*

# Tech stack

- Vite
- React 19
- tailwindcss v4
